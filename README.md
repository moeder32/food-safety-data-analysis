# food-safety-data-analysis
Power BI dashboard analyzing microbial contamination in tomato paste production, from TPC monitoring to external lab identification of bacteria 

## Overview
This project started as a tomato paste shelf-life trial aimed at understanding product longevity and improving shelf life.
---
##  Objectives
- Monitor microbial levels (TPC, Entro, Lacto)
- Identify high contamination cases
- Track pass vs retest outcomes
- Support escalation for external testing
- Analyze identified microorganisms
---
##  Process
- **Internal Testing:** TPC, Entro, Lacto used to monitor contamination  
- **Observation:** TPC showed consistently higher CFU levels  
- **Escalation:** High TPC triggered external lab testing  
- **External Results:** Bacillus identified as dominant microorganism  
---
##  Visual Approach
- **Matrix:** Compare test types across days  
- **Column Chart:** Highlight highest contamination (TPC)  
- **Line Chart:** Show trends over time  
- **Pie Chart:** Pass vs retest distribution  
- **Cards:** Key metrics summary  
- **Bar Chart:** External bacteria comparison  
---
##  Key Insights
- TPC is consistently higher than Entro and Lacto  
- High TPC acts as a trigger for further investigation  
- Bacillus is the primary contaminant  
- Low E. coli and Salmonella suggest process-related contamination
